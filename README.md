# MajokaIrisRssReader_ESP32

* ESP32 Yahoo RSS Reader
    * Auto Connect WiFi via WiFiManager.
    * Show RSS feed from Yahoo! Japan News.(Refresh every 3 min.)

# Hardware
* ESP-WROOM-32D
* Majoka IRIS LCD(640x48)
* FPC-24P 0.5mm adapter
* 5V Battery
